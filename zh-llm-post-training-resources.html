1. 大事记与脉络（2024-2026）

要理解 2024-2026 年中文社区围绕 LLM post-training 的讨论，先把握主线脉络： PPO → DPO 简化路线 → DeepSeek-R1/GRPO 引爆推理模型 → DAPO/Dr. GRPO/GSPO 等改进 → Agentic RL 兴起 → RL scaling 可预测化与 on-policy 蒸馏（2025Q4 起）。 几乎所有精华内容都围绕这一演化展开。
  • 2024 上半年：DPO 因实现简洁取代部分 PPO 场景；Iterative DPO、RLOO 等"轻量化 RLHF"百花齐放（同一脉络的 REINFORCE++ 实际发布于 2025.01，arXiv:2501.03262，并不属于这一时期）。
  • 2024.02：DeepSeek-Math 论文首次提出 GRPO，去除 Value Model，组内相对优势估计。
  • 2025.01：DeepSeek-R1 与 R1-Zero 发布，验证纯 RL 激发推理能力，引爆中文社区"推理模型"讨论。
  • 2025.03：字节&清华开源 DAPO，提出 Clip-Higher、动态采样、token-level loss、Overlong Reward Shaping 四项改进。
  • 2025.07：阿里 Qwen 团队发布 GSPO 论文（arXiv:2507.18071），将优化粒度从 token 提升到 sequence，解决 MoE 训练崩溃问题；注意时序：论文晚于 Qwen3 首发（2025.04），官方表述是 GSPO 助力了"最新一批 Qwen3 模型"的改进。
  • 2025-2026：Agentic RL 成为新主战场，多轮工具使用、长 horizon、ToolRL/RAGEN/TORL 等方法密集涌现。
  • 2025.09：DeepSeek-R1 登上《Nature》封面，披露 R1 的 RL 后训练阶段成本约 29.4 万美元——该数字不含其所基于的 V3 base 模型约 560 万美元的预训练成本，引用时须区分口径。
  • 2025.10：Meta 等发布 ScaleRL（arXiv:2510.13786），40 万+ GPU-hours 实验确立 RL 算力-性能曲线为 sigmoid 形、可由小规模训练外推；同月 Thinking Machines 博客提出 on-policy distillation，以约 1/10 的 RL 算力达到同级数学推理性能。
  • 2025.11：月之暗面开源 Kimi K2 Thinking（单任务可连续调用工具 200-300 次）；Ai2 发布 Olmo 3，开放 SFT→DPO→RLVR 全流程与 Dolci 后训练数据套件。
  • 2025.12：Prime Intellect 发布 INTELLECT-3（106B-A12B，arXiv:2512.16144），验证"开源 RL 环境生态（Environments Hub）+ 异步 RL 框架（prime-rl）"的社区化路线。
  • 2026.02：开源旗舰密集发布——智谱 GLM-5（744B-A40B，arXiv:2602.15763，slime 异步 RL 基础设施）、月之暗面 Kimi K2.5（2026.01.27 发布模型、02 月出技术报告 arXiv:2602.02276，文本-视觉联合 RL 与 Agent Swarm）、阿里 Qwen3.5-397B-A17B（02.16 上线）。
  • 2026.04：DeepSeek-V4 技术报告发布：以 10+ 领域 specialist（各自 SFT+GRPO）经全词表 logit 蒸馏（on-policy distillation）合并为单一模型、替代混合 RL，并以 DSA 稀疏注意力支撑原生 1M 上下文。

1.1 前沿补遗：2025Q4-2026H1 技术报告与一手资源

以下条目补齐 2025 年 10 月之后的进展。中文二手解读尚在沉淀，优先列一手技术报告与官方入口（链接已逐条核实）。
The Art of Scaling Reinforcement Learning Compute for LLMs（ScaleRL）2025.10
Meta 等首个 40 万+ GPU-hours 级 RL scaling 系统研究：RL 算力-性能曲线是 sigmoid 形（非 power law），可用小规模训练段外推渐近上限；并发现损失聚合、课程等设计选择主要影响算力效率而非最终渐近性能。把 RL 调参从"炼丹"推向可预测工程。
Thinking Machines：On-Policy Distillation2025.10
Kevin Lu 执笔：学生自己采样、教师逐 token 以反向 KL 打分，兼得 RL 的 on-policy 相关性与蒸馏的稠密监督。实测 Qwen3-8B-Base 在 AIME'24 达到 70% 仅需约 1800 GPU-hours（同效果 RL 需 17920），约 1/10 成本，是 2026 年 OPD 路线的起点。
Kimi K2 Thinking（月之暗面，开源）2025.11
K2 系列的 thinking agent 版本：边推理边用工具，单任务可连续调用工具 200-300 次，在 HLE、BrowseComp 等基准上取得开源 SOTA。Kimi 开放平台（platform.kimi.com）有思考模型的官方中文使用文档。
Olmo 3（Ai2）2025.11
7B/32B 全开放模型族（Base/Think/Instruct/RL Zero 多条路径），SFT→DPO→RLVR 三段后训练流水线连同 Dolci 数据套件全部开源，是目前最透明、最可复现的后训练参照系。
INTELLECT-3 技术报告（Prime Intellect）2025.12
106B-A12B MoE，用开源 prime-rl 异步 RL 框架（单机扩到数千 GPU、RL 阶段 512 张 H200）+ 社区 Environments Hub 环境平台训练，数学/代码/科学/推理同尺寸领先，验证了"众包 RL 环境 + 异步 RL"对抗大厂环境垄断的路线。
GLM-5: from Vibe Coding to Agentic Engineering（智谱）2026.02
744B-A40B 开源旗舰，完整披露后训练基础设施：自研 slime 异步 RL 框架将生成与训练解耦、配套异步 agent RL 算法，主打真实软件工程任务。代码与权重见 GitHub（zai-org/GLM-5）。
Kimi K2.5: Visual Agentic Intelligence（月之暗面）2026.02
文本-视觉联合预训练 + "zero-vision SFT"（纯文本微调激活视觉推理）+ 文本-视觉联合 RL，两模态互相增益；Agent Swarm 自指挥并行编排框架把复杂任务分解给并发子代理，延迟最高降 4.5 倍。
Qwen3.5-397B-A17B 上线（IT之家中文报道）2026.02
2026.02.16 阿里上线 Qwen3.5-Plus 与 Qwen3.5-397B-A17B，后者定位"Qwen3.5 开源系列旗舰大语言模型"：397B 总参、每次前向仅激活 17B，主打推理、编程、智能体与多模态。
DeepSeek-V4 / V4-Pro（Hugging Face 模型页）2026.04
后训练流水线最激进的改写：数学/代码/agent/指令各领域独立训 specialist（各自 SFT+GRPO），再经全词表 logit 的 on-policy 蒸馏合并为单一学生模型，整体替代混合 RL；DSA 稀疏注意力支撑原生 1M 上下文。

10. 推荐学习路径

下面给出一条"从 0 入门到能跑通自研 RLHF/GRPO"的中文路径，每个阶段都尽量使用前面提到的精华资源。

阶段 1：补齐基础（1-2 周）

  • 读张奇老师《大规模语言模型：从理论到实践》前 5 章，建立 LLM 全栈认知。
  • 看 B 站李沐 Transformer 论文精读 + happy-llm 教程，跑通 SFT。
  • 跟着 CS224R 中文笔记快速过一遍 RL 基础（策略梯度、PPO、value-based）。

阶段 2：RLHF 主线（2-3 周）

  • 读 Nathan Lambert "RLHF Book" 中文版，建立完整术语系统。
  • 精读知乎 "RLHF 全链路揭秘" + 图解 PPO 系列，对照 OpenRLHF 源码。
  • 动手：用 OpenRLHF 或 verl 跑通 PPO + DPO 各一次，对比训练曲线。

阶段 3：推理模型与 GRPO（2-3 周）

  • 读 DeepSeek-Math 原论文 + 知乎多篇中文解读，重点理解 GRPO 与 PPO 的差异。
  • 读 DeepSeek-R1 技术报告中文版，关注冷启动 + 多阶段训练流程。
  • 动手：用 verl 跑通 GRPO 训练（数学/代码任务），观察 entropy 与 reward 曲线。

阶段 4：前沿改进（持续）

  • 跟踪 DAPO / Dr. GRPO / GSPO / LitePPO 等改进，在自己的训练任务上做对比实验。
  • 读 Agentic RL 综述系列，把工具调用、长 horizon 等纳入训练管线。
  • 订阅 kexue.fm、机器之心、量子位、PaperWeekly，每周固定时间扫读新论文。
